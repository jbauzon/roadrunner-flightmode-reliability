# Changelog

## 5.0.0

First public release.

- React + WebSocket web GUI
- Full Pandion IBIT state machine (OFF -> OPERATE -> PLAYBACK -> IBIT)
- Flight Profile Playback mode (100 Hz, dual CSV format support)
- SITL simulator (firmware-accurate Pandion dialect 102)
- 6-UUT round-robin batch execution
- Debug Mode for manual vehicle commands
- SPDX license headers, Apache 2.0
- Python package: `rr_test` (pip install -e .)
